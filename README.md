Тестовое задание для компании CS-CART с использованием React, Typescript, Vite, react-hook-form<br/>
<br/>
Допустипые типы банковских карт при валидации: Visa, Mastercard<br/>
Допустимый диапазон значений окончания месяца банковской карты: 00 - 12<br/>
Допустимый диапазон значений окончания года банковской карты: 00 - 99<br/>
Максимальная длина имени и фамилии держателя карты: 128 символов<br/>
Допустимый тип ввода держателя: буквы, пробел<br/>
Допустивый диапазон значений CVC: 0 - 999<br/>
<br/>
Пример валидной карты: 5469 1234 1234 1234<br/>
<br/>
Данные о товарах выводятся на основе массива в App.tsx, сумма товаров суммируется и отображается на кнопке оплаты (вместе с комиссией за сервис)<br/>
<br/>
После успешной валидации и нажатого чекбокса показывает alert с даннными карты и ценой
